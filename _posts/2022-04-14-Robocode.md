# 1.什么是Robocode

- Robocode 是一个编程游戏，其目标是开发一个机器人战车来与 Java 或 .NET 中的其他坦克作战，并且机器人战斗在屏幕上实时运行
- Robocode带有一组预制的对手，可以帮助您入门，除此之外还可以与世界上最好的联盟对抗
- 每个Robocode参与者都可以使用Java语言的元素来创建自己的机器人，从而使各种开发人员都可以参与其中
